CSS Selectors

There are three main ways to use selectors in CSS

  1. Attribute
  2. ID
  3. Class

By attribute:

The attribute selector follows the Attribute[value] format and looks for attributes with matching values to add to the style, an example is a[target="blank"] which will change any links to the specified styles.

By ID:

ID is one of the most specific ways to use CSS and can be called up using #. For example, a div, section, or header element can be given an ID as so. <div id="introduction"> tag and can be called up in css using:

#introduction {
  font-family: Helvetica;
}

By class:

A class can be specified by appending class to the div, section, or other element as such:
<section class="introduction">
